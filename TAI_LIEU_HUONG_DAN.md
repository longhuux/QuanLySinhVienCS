# 📖 Tài liệu Hướng dẫn - Tổng hợp

Chào mừng bạn đến với **Hệ thống Quản lý Sinh viên**! 

Dưới đây là danh sách các tài liệu hướng dẫn có sẵn, được sắp xếp theo mức độ chi tiết và mục đích sử dụng.

## 🎯 Bạn nên đọc file nào?

### 👉 Nếu bạn là người mới bắt đầu:
**Đọc:** [`QUICK_START.md`](QUICK_START.md)
- Hướng dẫn nhanh trong 5 phút
- Các thao tác cơ bản nhất
- Ví dụ thực tế từng bước

### 👉 Nếu bạn cần hướng dẫn chi tiết:
**Đọc:** [`DOCUMENTATION.md`](DOCUMENTATION.md)
- Tài liệu đầy đủ và chi tiết nhất
- Hướng dẫn từng tính năng
- Xử lý lỗi và FAQ
- Mẹo và thủ thuật

### 👉 Nếu bạn cần hướng dẫn bằng tiếng Việt ngắn gọn:
**Đọc:** [`HUONG_DAN_SU_DUNG.md`](HUONG_DAN_SU_DUNG.md)
- Hướng dẫn ngắn gọn, dễ hiểu
- Tập trung vào các chức năng chính
- Công thức tính điểm

### 👉 Nếu bạn muốn biết về cấu trúc dự án:
**Đọc:** [`README.md`](README.md)
- Kiến trúc 3 lớp
- Cấu trúc thư mục
- Công nghệ sử dụng
- Hướng dẫn build và chạy

### 👉 Nếu bạn cần mẫu file Excel:
**Đọc:** [`SampleData/StudentTemplate.md`](SampleData/StudentTemplate.md)
- Cấu trúc file Excel để import
- Ví dụ dữ liệu mẫu
- Lưu ý khi import

1. ✅ Đọc `QUICK_START.md`

2. ✅ Thử tính năng Import/Export Excel
3. ✅ Sử dụng bộ lọc nâng cao
4. ✅ Xem Dashboard và cảnh báo học vụ
